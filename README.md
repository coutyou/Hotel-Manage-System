## 开发工具
* QT 5.10.1
* QT Creator 4.6.1(community)
* SQLite Developer Lite
* StarUML(用于反向生成UML图)

## 正确的打开方式
1. 安装两个字体文件
2. 打开release文件夹中的HotelManageSystem.exe运行程序
